Inline vs Block-level Elements

Block-level Elements

This is a block-level element (e.g., <div>).

This is another block-level element (e.g., <p>).

Heading tags like <h1>, <h2>, etc., are also block-level.

Inline Elements

This paragraph contains inline elements: This is a <span>. This is a <a> (link). This is <strong>. This is <em>.

Notice how inline elements stay within the flow of text and do not start on a new line.

Visual Differences

Block elements (blue boxes) always start on a new line and take up the full width of the container.

Inline elements (red dashed boxes) stay within the line and take up only as much width as their content.
